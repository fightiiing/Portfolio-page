Superhero Street Fight!

A single page React app with parallax scrolling where you choose two superhero characters to fight. The character stats are pulled from an api, which is then used when the FIGHT button is clicked. The code first determines if an "attack" hits the opponent. If it does, then based on the attacking stats and the defensive stats of the opponent, a damage number is calculated, normalized, and then subtracted from the opponent's health.
